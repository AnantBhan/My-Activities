What is it?

This is the starter code for CS390MB Mobile Health Sensing. My Activities is a daily health monitoring application, which tracks personal fitness, heart rate, sociability and frequent locations. The framework is provided; be ambitious!

Getting Started

Clone the repository, then follow the instructions for assignment A0 to live-stream your accelerometer data on the web.

Assignment A1 : Step Detection

You will design a step detection algorithm and implement it both locally in the Android code and in a Python script. See the assignment details for more.
